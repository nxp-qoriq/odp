odp_lso_protocol_t odp_lso_profile_param_t::lso_proto

Segmentation protocol.

Selects on which protocol LSO operation performs segmentation (e.g. IP fragmentation vs. TCP segmentation). When ODP_LSO_PROTO_CUSTOM is selected, only custom field modifications are performed. The default value is ODP_LSO_PROTO_NONE. Check LSO capability for supported protocols.

odp_lso_modify_t odp_lso_profile_param_t::mod_op

Field modify operation.

Selects how value of the field is modified from its original value during segmentation. Field value is assumed to be in network (big endian) byte order.
